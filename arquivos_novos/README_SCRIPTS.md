```

4. Pressiona ENTER quando pedir

```

```

3. Copia e cola as rotas (está no output do script)
4. Salva
5. Roda: pnpm dev
6. Testa: http://localhost:3001
7. Push: git push
```

## 🆘 SE DER ERRO

**Erro: "Pasta não encontrada"**
- Cria pasta `arquivos_novos/` na RAIZ
- Descompacta o ZIP lá
- Roda o script novamente

**Erro: "Arquivo não encontrado"**
- Verifica se o ZIP foi descompactado completamente
- Tira espaços do nome da pasta
- Roda o script de novo

**Erro: "Acesso negado"**
- Fecha VS Code
- Roda o script como ADMIN
- Clica direito no .bat → "Executar como administrador"
